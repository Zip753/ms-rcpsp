Detailed Description

Base abstract class for specimen (schedule) representations.

If you want to implement your own representation of schedule, you must create a subclass of Schedule and implement Schedule::compute_fitness function. Then you have to instantiate all Crossover and Mutator operators you're going to use for your new subclass. For instantiation example see src/UniformCrossover.cpp.
